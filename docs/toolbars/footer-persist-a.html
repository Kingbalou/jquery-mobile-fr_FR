Cette page est une démo d'une barre de navigation de pied de page persistante. Au pied de la page, vous verrez une barre de navigation horizontale persistante. Cliquez sur un des liens, et vous verrez la transition du contenu de la page, mais le pied de page reste fixe : Les pieds de page reste persistants, même si c'est une transition vers une nouvelle page HTML, parce que le pied de page sur les quatre pages HTML a le même attribut data-id . Remarque : Si vous souhaitez un bouton actif dans votre barre de navigation qui reste actif lorsque vous revenez à la page affichant ce bouton actif, ajouter une classe de ui-state-persist, en plus de ui-btn-active à l'ancrage correspondant.
