One

J'ai un id "one" sur ma page conteneur. Je suis la première dans l'ordre de source, donc je suis affiché lorsque la page se charge.

C'est un modèle passe-partout multi-pages que vous pouvez copier pour vous construire la première page jQuery Mobile. Ce modèle contient à l'intérieur des conteneurs de plusieurs "page", contrairement à un modèle de page unique qui n'a qu'une d'une page à l'intérieur.

Affichez juste la source et copiez le code pour démarrer. Toutes les CSS et le JS sont liés aux versions CDN de jQuery donc c'est super facile à configurer. N'oubliez pas d'inclure une balise meta viewport dans le head pour fixer le niveau de zoom.

Vous liez des pages internes en se référant à l'ID de la page que vous voulez montrer. Par exemple, pour le lien à la page avec un ID "two", mon lien aura un href="#two" dans le code.
