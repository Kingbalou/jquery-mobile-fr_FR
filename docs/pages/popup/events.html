Le plugin popup a les événements personnalisés suivants :

opened déclenché quand un popup est ouvert

$( ".selector" ).popup({
   opened: function(event, ui) { ... }
});

Cet événement est déclenché après que le popup soit complètement affiché sur l'écran, ce qui signifie que toutes les animations associées soient terminées.

closed déclenché quand un popup est fermé

$( ".selector" ).popup({
   closed: function(event, ui) { ... }
});

Cet événement est déclenché après que le popup soit complètement masqué de l'écran, ce qui signifie que toutes les animations associées soient terminées.
