Le plugin popup a les méthodes suivantes :

open(x, y, $link)affiche la popup centrée aux coordonnées (x, y) tilisant le lien spécifié comme origine

$('.selector').popup('open', x, y, $link);

Si x ou y n'est pas renseigné, le milieu de la fenêtre sera utilisée. $link est optionnelet il spécifie le lien qui permet l'ouverture du popup. $link peut influencer le comportement de la popup via ses attributs data-position-to et data-transition.

close ferme un popup ouvert

$('.selector').popup('close');
