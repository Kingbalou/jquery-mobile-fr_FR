Pour transformer n'importe quel div en popup, ajoutez l'attribut data-role="popup" au div. Si vous voulez que le popup apparaisse quand l'utilisateur clique sur un lien, créer un lien de telle sorte que son attribut href soit défini sur l'id du popup et le lien a l'attribut data-rel="popup". 


<a href="#popupBasic" data-rel="popup">Tooltip</a>

<div data-role="popup" id="popupBasic">
	Il s'agit d'un popup de base, sans options définies.
</div>

Cela se traduira par ce qui suit : popup. Notez que vous n'avez pas besoin d'espacement sur le popup car les widgets l'adapteront proprement.

Les fenêtres des popup peuvent avoir diverses transitions :

Pas de transition Pop Slide up Slide down

Ceci est un texte pour l'exemple de transition

Vous pouvez ajouter un attribut data-transition au popup pour lui attribuer une autre transition que celle par défaut $.mobile.defaultDialogTransition. Cette affectation peut alors être remplacée par un attribut data-transition spécifié sur le lien qui ouvre le popup. Les exemples de transition ont le code suivant : 


<a href="#transitionExample" data-transition="none" data-role="button" data-inline="true" data-rel="popup">Pas de transition</a>
<a href="#transitionExample" data-transition="pop" data-role="button" data-inline="true" data-rel="popup">Pop</a>
<a href="#transitionExample" data-transition="slideup" data-role="button" data-inline="true" data-rel="popup">Slide up</a>
<a href="#transitionExample" data-transition="slidedown" data-role="button" data-inline="true" data-rel="popup">Slide down</a>
<div id="transitionExample" data-role="popup" data-transition="flip">
  <p>Ceci est un texte pour l'exemple de transition</p>
</div>

Lorsque vous lancez le popup depuis l'un des boutons, la data-transition donnée pour ce bouton sera utilisée. Toutefois, si vous lancez le popup par d'autres moyens, tels que, par exemple, via $("#transitionExample").popup("open"), la data-transition donnée pour le popup sera utilisée (la transition flip).

Appel du plugin popup

Ce plugin sera auto-initialiser sur n'importe quelle page qui contient un div avec l'attribut data-role="popup". Cependant, si nécessaire, vous pouvez appeler directement le plugin popup sur n'importe quel sélecteur, comme n'importe quel plugin jQuery :


$('#myPopupDiv').popup();

Thématisation du popup

Le plugin popup fournit 2 options de thème : data-theme et data-overlay-theme. L'option data-theme fait référence au thème de la popup, alors que data-overlay-theme se réfère au thème du fond de la popup, qui couvre toute la fenêtre derrière la popup.

data-theme héritera de la page et aura toujours une valeur valide lorsque le popu s'ouvrira, à moins que vous spécifiez explicitement data-theme="none", dans ce cas, la popup n'aura pas du tout de thème.

Le data-overlay-theme ne sera jamais défini et le popup de fond, bien qu'il soit toujours présent lorsque la popup s'affiche, sera complètement transparent, à moins explicitement de le définir en utilisant par exemple data-overlay-theme="a". Dans ce cas, l'arrière-plan sera fade in, cachant partiellement le reste de la fenêtre, pour attirer l'attention sur la popup Voici un exemple d'une popup thématisé :


<a href="#both" data-rel="popup" data-role="button">Thème + recouvrement définis</a>
<div id="both" data-role="popup" data-overlay-theme="a" data-theme="e" class="ui-content">
  <p>J'ai <code>data-overlay-theme="a"</code> et <code>data-theme="e"</code></p>
</div>

Recouvrement défini à A
