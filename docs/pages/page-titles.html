Titres dans la navigation Ajax

Lorsque vous chargez la première page d'un site basé sur jQuery Mobile, puis que vous cliquez sur un lien ou soumettez un formulaire, Ajax est utilisé pour tirer le contenu de la page requêtée. Avoir les deux pages dans les DOM est essentielle pour permettre les transitions de page animées, mais un inconvénient sur cette approche est que le titre de la page est toujours celle de la première page, et non celle de la page suivante que vous visualisez.

Pour remédier à cela, jQuery mobile analyse automatiquement le title de la page tirée via AJAX et change l'attribut title du document parent pour le faire correspondre.

Titres dans les modèles multi-page

Sur des documents multi-page, nous suivons une convention similaire, mais toutes les pages partagent untitle commun, nous avons un attribut data-title qui peut être ajouté à chaque conteneutr de page dans un modèle multi-page pour définir manuellement un titre. Le titre du document HTML sera automatiquement mis à jour pour correspondre au data-title de la page en cours à l'affichage.


<div data-role="page" id="foo" data-title="Page Foo">

</div><!-- /page -->
