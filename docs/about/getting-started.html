Débuter avec jQuery Mobile

jQuery Mobile fournit un ensemble de widgets d'interface utilisateur tactile conviviale et un système de navigation AJAX pour supporter les transitions de page animée. Construire votre première page jQuery Mobile c'est facile, voici comment faire :

Créer un modèle de page de base

Ouvrez votre éditeur de texte favori et collez ce qui suit dans le modèle de page, sauvez et ouvrez le dans un navigateur. Vous êtes maintenant un développeur mobile !

Voici ce qu'il y a dans le modèle. Dans le head, un méta tag viewport définit la largeur de l'écran à celle de la largeur en pixel de l'appareil, des références à jQuery, à jQuery Mobile, à la feuille de style du thème du mobile depuis le CDN, et tous les styles et les scripts.

Dans le body, un div avec un data-role de page est l'enveloppe utilisée pour délimiter une page, la barre d'entête (data-role="header") et la région du contenu (data-role="content") sont ajoutés à l'intérieur pour créer une page de base (ceux-ci sont facultatifs tous les deux facultatifs). Ces attributs data- sont des attributs HTML5 qui sont utilisés à travers jQuery Mobile pour transformer un balisage de base dans un widget amélioré et stylisé.


<!DOCTYPE html> 
<html> 
	<head> 
	<title>Ma Page</title> 
	<meta name="viewport" content="width=device-width, initial-scale=1"> 
	<link rel="stylesheet" href="http://code.jquery.com/mobile/1.0/jquery.mobile-1.0.min.css" />
	<script src="http://code.jquery.com/jquery-1.6.4.min.js"></script>
	<script src="http://code.jquery.com/mobile/1.0/jquery.mobile-1.0.min.js"></script>
</head> 
<body> 

<div data-role="page">

	<div data-role="header">
		<h1>Mon Titre</h1>
	</div><!-- /header -->

	<div data-role="content">	
		<p>Hello le monde</p>		
	</div><!-- /content -->

</div><!-- /page -->

</body>
</html>

Ajoutez votre contenu

À l'intérieur de votre conteneur de contenu, vous pouvez ajouter tous les éléments HTML standard - titres, listes, paragraphes, etc. Vous pouvez écrire vos propres styles personnalisés pour créer des mises en page personnalisées en ajoutant une feuille de style personnalisée au head après la feuille de style de jQuery Mobile.

Faire une liste

jQuery Mobile comprend un ensemble diversifié de listes communes qui sont codées sous forme de listes avec un data-role="listview" ajouté. Voici une simple liste liée qui a un rôle de listview. Nous allons faire afin que cela ressemble à un module en ajoutant un data-inset="true" et nous ajouterons un filtre de recherche dynamique avec l'attribut data-filter="true".


<ul data-role="listview" data-inset="true" data-filter="true">
	<li><a href="#">Acura</a></li>
	<li><a href="#">Audi</a></li>
	<li><a href="#">BMW</a></li>
	<li><a href="#">Cadillac</a></li>
	<li><a href="#">Ferrari</a></li>
</ul>

  • Audi
  • BMW
  • Cadillac
  • Ferrari

Ajouter un curseur

Le framework contient un ensemble complet d'éléments de formulaire qui sont automatiquement améliorés en widgets tactiles conviviales stylisés. Voici un curseur fait avec un nouvel input HTML5 de type range, il n'y a pas besoin de data-role. Assurez vous d'envelopper ceux-ci dans un élément form et toujours bien associer un label pour chaque élément du formulaire.


<form>
   <label for="slider-0">Curseur :</label>
   <input type="range" name="slider" id="slider-0" value="25" min="0" max="100"  />
</form>

Faire un bouton

Il existe plusieurs façons de faire des boutons, mais la possibilité de transformer un lien en bouton est donc plus facile à cliquer. Il suffit de commencer avec un lien et de lui ajouter un attribut data-role="button". Vous pouvez ajouter une icône avec l'attribut data-icon et éventuellement définir sa position avec l'attribut data-iconpos.


<a href="#" data-role="button" data-icon="star">Bouton étoile</a>
Bouton étoile

Jouer avec les échantillons de thème

jQuery Mobile a un framework de thème robuste qui supporte jusqu'à 26 ensembles de couleurs de barre d'outils, de contenu et de bouton, appelé "Swatch" (Echantillon). Ajoutez juste l'attribut data-theme="e" à n'importe quels widgets sur cette page : page, entête, liste, curseur ou bouton pour le passer en jaune. Essayez les différentes lettres (de a à e) d'échantillon dans le thème par défaut pour mélanger et assortir les échantillons.

Astuce intéressante : ajoutez l'échanillon du thème à la page et regardez comment tous les widgets à l'intérieur du contenu hériteront automatiquement du thème (les entêtes n'héritent pas, ils sont par défaut avec l'échantillon A).

<a href="#" data-role="button" data-icon="star" data-theme="a">Button</a> data-theme="a" data-theme="b" data-theme="c" data-theme="d" data-theme="e"

Lorsque vous êtes prêt à créer un thème personnalisé, utilisez ThemeRoller pour glisser et déposer, puis télécharger un thème personnalisé.

Pour continuer et construire des chose

Ce n'est qu'effleurer la surface de toutes les choses cool que vous pouvez construire avec jQuery Mobile sans trop d'effort. N'oubliez pas d'explorer la liaison de pages, l'ajout des transitions de page animées et la création des boîtes de dialogue. Utilisez la référence data-attribute afin de pouvoir jouer avec d'autres attributs data-.

Plus d'infos du développeur ? Bien, oubliez tout ce que nous venons de voir (blague). Si vous ne voulez pas utiliser le système de configuration de l'attribut data-, vous pouvez prendre le contrôle complet de tout et appelez les plugins directement parce que ce sont simplement tous des plugins jQuery standard intégré avec l'ensemble des widgets de l'interface utilisateur. Veillez à creuser dans la configuration globale, les événements et les méthodes.
