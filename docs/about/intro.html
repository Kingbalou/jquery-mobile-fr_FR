Aperçu de jQuery Mobile

La stratégie de jQuery Mobile peut être résumée simplement : un système d'interface utilisateur unifié qui fonctionne parfaitement sur toutes les plateformes des appareils mobiles les plus populaires et qui est construit sur les fondations solides de jQuery et jQuery UI. Axée sur un code léger, construit sur l'amélioration progressive avec une conception flexible et facilement personnalisable.

La différence essentielle avec notre approche est la grande variété de plateformes mobiles visée avec jQuery Mobile. Nous avons travaillé dur pour apporter un support jQuery à tous les navigateurs mobiles qui sont suffisamment compatibles et qui ont un minimum de parts de marché. De cette façon, nous traitons les navigateurs web de mobile exactement comme nous traitons les navigateurs web de bureau.

Pour rendre ce support aussi large que possible, toutes les pages dans jQuery Mobile sont construites sur une base propre, la sémantique HTML, pour assurer la compatibilité avec quasiment n'importe quel appareil compatible Web. Parmi les dispositifs qui interprètent le CSS et le JavaScript, jQuery Mobile applique des techniques d'amélioration progressive pour transformer discrètement la page sémantique dans une expérience riche et interactive qui exploite la puissance de jQuery et CSS. Les fonctions d'accessibilité telles que WAI-ARIA sont étroitement intégrées dans le framework pour fournir un support pour les lecteurs d'écran et les autres technologies d'assistance.
