Listes pliables

Accueil Recherche

Les listes peuvent être utilisées conjointement avec le widget collapsible. Le framework les adapte parfaitement à l'intérieur du conteneur de contenu généré pour le pliable. Si un data-content-theme a été défini (qui ajoute une bordure au conteneur de contenu), le framework supprime la bordure inférieure du dernier élément de la liste et ajuste le style de ses coins.
