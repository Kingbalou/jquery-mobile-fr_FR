Formatage du contenu

Accueil Recherche

Le contenu des pages en jQuery Mobile est complètement ouvert, mais le framework jQuery Mobile fournit un certain nombre d'outils utiles et des widgets - tels que les panneaux pliables et les grilles de mise en page multi-colonne - pour faciliter le formatage de votre contenu pour les appareils mobiles.

  • Styles HTML de base
  • Grille de mise en page (colonnes)
  • Blocs de contenu pliable
  • Ensembles pliable (accordéons)
  • Thématisation du contenu
