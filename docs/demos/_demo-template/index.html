Comment fonctionne "voir le code source"

C'est un div avec ...

data-demo-html="true" pour inclure le balisage dans ce div dans la vue du code source.

data-demo-js="true" pour inclure tous les scripts depuis l'entête dans la vue du code source.

data-demo-css="true" pour inclure tous les styles depuis l'entête dans la vue du code source.

Spécifiez quel balisage, quel script ou quel style pour afficher

C'est un div avec ...

data-demo-html="#myMarkup" pour inclure un balisage spécifique dans la vue du code source.

data-demo-js="#myScript" pour inclure un script spécifique depuis l'entête dans la vue du code source.

data-demo-css="#myStyle" pour inclure un style spécifique depuis l'entête dans la vue du code source.

data-demo-php="source.php" pour inclure le script PHP depuis source.php, qui est la version en texte brut du fichier file-name.php, dans la vue du code source (voir les fichiers dans ce dossier).
