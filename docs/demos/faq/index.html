Questions & Réponses

Les choses ne fonctionnent pas toujours comme cela est expliquer dans la documentation, nous avons recueilli des questions courantes et des solutions pour vous aider à devenir rapidement opérationnel et résoudre des problèmes délicats lorsque l'on travaille avec jQuery Mobile.

  • Pourquoi mes styles CSS ne s'appliquent pas correctement à un widget ?

  • Mon type d'entrée range / search est changé en number / text.
  • Parfois ma sélection personnalisée s'affiche comme une popup et d'autres fois c'est une boîte de dialogue.
  • La désactivation d'un bouton ne fonctionne pas.
  • Les contrôles dans mon entête ou mon pied de page fixe ne répondent pas ou se comportent de façon erratique.

  • J'essaie le cache de l'application, mais ça ne marche pas ?
  • Le contenu injecté dans une page n'est pas amélioré.
  • Le déclechement de create sur du HTML injecté ne marche pas.
