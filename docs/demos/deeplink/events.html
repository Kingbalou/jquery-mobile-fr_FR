Liez les événements directement à l'élément input. Utilisez les évènements virtuels de jQuery Mobile ou liez les évènements standard de JavaScript tel que change, focus, blur, etc... :


$( ".selector" ).bind( "change", function(event, ui) {
  ...
});

Le plugin curseur a les événements personnalisés suivants :

create triggered when a slider is created

$( ".selector" ).slider({
   create: function(event, ui) { ... }
});
slidestart déclenché quand il y a une interaction initiale avec le curseur. Inclure les drags et taps.

$( ".selector" ).on( 'slidestart', function( event ) { ... });
slidestop déclenché à la fin d'une interaction avec le curseur. Inclure les drags et taps.

$( ".selector" ).on( 'slidestop', function( event ) { ... });
