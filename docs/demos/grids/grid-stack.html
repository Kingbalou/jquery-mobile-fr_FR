Grille Responsive

C'est facile de prendre les grilles standard et les rendre responsive. Dans cet exemple, nous voulons que tous les blocs de la grille s'empilent sur des largeurs étroites. Au dessus de 40em (640 pixels) de large, nous voulons retrouver les styles des grilles multi-colonnes.

Nous recommandons normalement l'écriture de point d'arrêt utilisant min-width. Chaque points d'arrêt permettent de modifier progressivement la mise en page, du plus petit au plus large des écrans.

Cependant, dans cette situation, puisque nous voulons juste remplacer les positions et les largeurs des grilles standard sous un point d'arrêt unique, il est plus simple d'ajouter un point d'arrêt max-width qui n'appliquera le style empilé qu'en dessous d'une certaine largeur. Cela nous permet de profiter de tous les styles de grille par défaut au dessus de la largeur du point d'arrêt.

Pour faire que tout cela fonctionne, ajoutez une classe (my-breakpoint) au conteneur de la grille pour porter les styles empilés pour chaque classe de bloc à l'intérieur du point d'arrêt de la media query. Les styles au sein du point d'arrêt afin que tous les de blocs s'empilent de façon vraiment concise :


/* Empile toutes les grilles en dessous de 40em (640px) */
@media all and (max-width: 35em) {
	.my-breakpoint .ui-block-a, 
	.my-breakpoint .ui-block-b, 
	.my-breakpoint .ui-block-c,
	.my-breakpoint .ui-block-d,
	.my-breakpoint .ui-block-e { 
		width: 100%; 
		float:none; 
	}
}

C'est tout le code nécessaire à l'écriture d'un point d'arrêt personnalisé pour faire que tous les différents types de grilles soient responsive. A partir de cette base, vous pouvez plus personnaliser l'apparence ou même ajouter des points d'arrêt supplémentaires.
