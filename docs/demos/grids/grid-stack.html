Grille Responsive

C'est facile de prendre les grilles standard et les rendre responsive. Dans cet exemple, tous les blocs de grille sont empilés sur des largeurs étroites comme un smartphone. Au dessus de 36em (576 pixels) de largeur, les grilles côte à côte sont créées en positionnant les blocs et en paramétrant le pourcentage pour les largeurs.

Démo de grille Responsive

Chacune de ces grilles s'affiche comme un ensemble de blocs empilés sur des largeurs étroites.

Exemple de styles

Voici les styles dont vous avez besoin pour empiler toutes les grilles. Tous fonctionnent en ajoutant rwd-stack à chaque conteneur de grille.


/* Empile tous les blocs au départ */
.rwd-stack .ui-block-a, 
.rwd-stack .ui-block-b, 
.rwd-stack .ui-block-c,
.rwd-stack .ui-block-d,
.rwd-stack .ui-block-e { 
	width: 100%; 
	float:none; 
}

/* Positionne et définit des largeurs au dessus de 576px (36em * 16px) */
@media all and (min-width: 36em){
	/* Positionne tous les blocs */
	.rwd-stack .ui-block-a, 
	.rwd-stack .ui-block-b, 
	.rwd-stack .ui-block-c,
	.rwd-stack .ui-block-d,
	.rwd-stack .ui-block-e { 
		float:left; 
	}
	/* Définit des largeurs jusqu'à 2 grilles */
	.rwd-stack.ui-grid-a .ui-block-a, 
	.rwd-stack.ui-grid-a .ui-block-b { 
		width: 49.95%; 
	}
	/* Définit des largeurs jusqu'à 3 grilles */
	.rwd-stack.ui-grid-b .ui-block-a, 
	.rwd-stack.ui-grid-b .ui-block-b, 
	.rwd-stack.ui-grid-b .ui-block-c { 
		width: 33.25%; 
	}
	/* Définit des largeurs jusqu'à 4 grilles */
	.rwd-stack.ui-grid-c .ui-block-a, 
	.rwd-stack.ui-grid-c .ui-block-b, 
	.rwd-stack.ui-grid-c .ui-block-c,
	.rwd-stack.ui-grid-c .ui-block-d { 
		width: 24.925%;
	}
	/* Définit des largeurs jusqu'à 5 grilles */
	.rwd-stack.ui-grid-d .ui-block-a, 
	.rwd-stack.ui-grid-d .ui-block-b, 
	.rwd-stack.ui-grid-d .ui-block-c,
	.rwd-stack.ui-grid-d .ui-block-d,
	.rwd-stack.ui-grid-d .ui-block-e { 
		width: 19.925%;
	}
}
