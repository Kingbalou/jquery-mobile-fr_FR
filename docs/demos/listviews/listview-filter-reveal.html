Autocomplete local avec un filtre de liste

La fonction de filtre permet de construire facilement une simple saisie semi-automatique avec des données locales. Lorsqu'une liste filtrable a data-filter-reveal="true", il masquera automatiquement tous les éléments de la liste lorsque le champ de recherche est vide. L'attribut data-filter-placeholder peut être ajouté pour spécifier le texte du placeholder pour le filtre.

Les filtres de liste avec plus de 100 à 200 éléments peuvent être lents sur un appareil mobile, donc nous vous recommandons d'utiliser cette fonction pour les situations d'auto-complétion avec un nombre relativement faible d'éléments. Si vous avez besoin d'une recherche dans une liste contenant plus de valeurs, nous fournissons un moyen de créer un filtre avec une source de données distante .
