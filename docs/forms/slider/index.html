Pour ajouter un widget curseur à votre page, utiliser l'input standard avec l'attribut type="range". L'entrée value est utilisé pour configurer la position de départ du curseur et la valeur est peuplée dans la zone de saisie. Précisez les valeurs des attributs min et max pour régler la fourchette du curseur. Le framework analysera ces attributs pour configurer le widget curseur. Regardez la reference de data- attribute pour voir tous les attributs possibles que vous pouvez ajouter aux curseurs.

Lorsque vous faites glisser le curseur, le framework met à jour la valeur de l'entrée natif (et vice-versa) afin qu'ils soient toujours en phase, ce qui garantit que la valeur est soumise avec le formulaire.

Réglez l'attribut for de label pour faire correspondre l'ID de l'input ainsi ils sont sémantiquement liés. Il est possible de cacher le label d'accessibilité si il n'est pas souhaitée dans la mise en page, mais nous avons besoin qu'il soit présent dans le balisage pour des raisons de sémantique et d'accessibilité.

Le framework trouvera tous les éléments input avec un type="range" et les améliorera automatiquement dans un curseur avec une saisie complémentaire, pas besoin d'appliquer l'attribut data-role. Pour éviter l'amélioration automatique de cette entrée dans un curseur, ajouter l'attribut data-role="none" à l'input et enveloppez les d'un div avec l'attribut data-role="fieldcontain" pour les grouper.


   <label for="slider-0">Curseur :</label>
   <input type="range" name="slider" id="slider-0" value="25" min="0" max="100"  />

Les curseurs répondent également à des commandes via des touches. Les touches flèche vers la droite, flèche vers le haut et haut de page augmentent la valeur, flèche gauche, flèche vers le bas, et bas de page pour la diminuer. Pour déplacer le curseur à sa valeur minimum ou maximum, utilisez respectivement la touche début et fin.

Appel du plugin curseur

Ce plugin sera automatiquement initialiser sur n'importe quelle page qui contient un input de texte avec l'attribut type="range". Cependant, si nécessaire, vous pouvez appeler directement le plugin curseur sur un sélecteur,comme n'importe quel plugin jQuery :


$('input').slider();

Thématisation du curseur

Pour définir l'échantillon de thème pour le curseur, ajoutez l'attribut data-theme à l'input qui appliquera le thème à la fois à la saisie, au curseur et à la piste. L'échantillon de la piste du curseur peut être réglé séparément en ajoutant l'attribut data-track-theme à appliquer à la version de l'état down de l'échantillon du bouton sélectionné.


<div data-role="fieldcontain">
	<label for="slider-2">Curseur :</label>
	<input type="range" name="slider-2" id="slider-2" value="25" min="0" max="100" data-theme="a" data-track-theme="b" />
</div>
