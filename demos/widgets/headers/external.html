API des entêtes externes

L'entête et le pied de page sur cette page sont positionnés en dehors de la page. Ils sont avant et après la page à l'intérieur du body. Ces barres d'outils resteront dans le DOM jusqu'à leur suppression manuelle. Si vous naviguez sur une page qui ne dispose pas d'une entête ou d'un pied de page ces barres d'outils persisteront. Sinon, ils vont se cacher pour permettre à l'entête et au pied de page de la page de s'afficher.

Balisage d'une entête

Le balisage d'entêtes externes est identique à des entêtes normales, vous placez simplement l'entête en dehors de la page dans le body de votre page

Auto Initialisation

Parce que ces barres d'outils ne sont pas dans la page, ils ne seront pas initialiser automatiquement. Vous devez appeler vous-même le plugin barre d'outils lorsque le document est prêt.

Navigation Ajax

Parce que ces barres d'outils ne sont pas dans la page, ils restent dans le DOM jusqu'à leur suppression manuelle. Cependant, ils vont se cacher automatiquement si vous accédez à une page contenant une barre d'outils pour permettre aux propres barres d'outils de prendre leur place. Ils ré-apparaîtront automatiquement si vous accédez à une page ne ​​contenant pas de barres d'outils.

Les barres d'outils en dehors d'une page ne seront pas tirées dans le DOM pendant la navigation Ajax.
