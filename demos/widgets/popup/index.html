Popup API

Le widget popup peut être utilisé pour différents types de popups. D'un petit tooltip à un album photo grand format.

Popup basiques

Pour créer un popup, ajoutez l'attribut data-role="popup" au div avec le contenu du popup. Ensuite créez un lien avec href défini avec l'id du div du popup et ajoutez l'attribut data-rel="popup" pour appeler le framework afin d'ouvrir le popup lorsqu'il sera touché. Un div de popup doit être imbriqué à l'intérieur de la même page que le lien.

Boîte de dialogue

Le balisage standard d'une boîte de dialogue peut être placé dans une popup. Pour créer une boîte de dialogue de style modal, ajouter l'attribut data-dismissible="false" à la popup pour empêcher le comportement du clic extérieur afin que les gens soient obliger d'interagir avec les boutons de la popup pour la fermer.

Ajout d'espacement

Pour les popups avec du texte, l'espacement est nécessaire. La classe ui-content peut être ajoutée à la popup pour ajouter un espacement standard de 15px. Lorsque l'espacement est ajouté, nous appliquons quelques règles de style pour annuler la marge supérieure de l'entête ou d'un paragraphe dans le popup et faisons de même pour la marge du bas du dernier élément.

Fermeture des popups

¨Par défaut, les popups peuvent être fermés soit en cliquant en dehors du widget popup, soit en pressant la touche Echap. Pour éviter cela, l'attribut data-dismissible="false" peut être ajouté à la popup. Pour ajouter un bouton de fermeture explicite à la popup, ajoutez un lien avec le rôle button dans le conteneur de la popup avec un attribut data-rel="back" une position via une classe.

Position

Par défaut, les popus s'ouvrent centrées verticalement et horizontalement sur ​​l'élément que vous avez cliqué (l'origine), ce qui est bien pour les popups utilisés comme info-bulles ou menus. Si une popup doit apparaître centrée dans la fenêtre à la place de l'origine, ajoutez l'attribut data-position-to au lien et spécifier la valeur window. Il est aussi possible de spécifier n'importe quel sélecteur valide comme valeur de position-to en plus de origin et window.

Transitions

Par défaut, les popups n'ont pas de transition pour s'ouvrir aussi rapidement que possible. Pour définir la transition utilisée par la popup, ajoutez l'attribut data-transition au lien qui référence la popup. La transition inverse sera utilisé lors de la fermeture de la popup. Pour des raisons de performances sur les appareils mobiles, nous recommandons d'utiliser des transitions simples comme pop, fade ou none pour des animations de popup lisse et rapide.

Thème

Vous avez alors deux options liées au thème : data-theme et data-overlay-theme. L'option data-theme fait référence au thème de la fenêtre elle-même, alors que data-overlay-theme contrôle la couche semi-opaque derrière la fenêtre contextuelle. Le thème est héritée de la page, spécifiez data-theme="none" pour une popup avec un fond transparent.
