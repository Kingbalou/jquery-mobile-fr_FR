Radio boutons API

Les radio boutons sont utilisées pour fournir une liste d'options où une seule option peut être sélectionnée.

Balisage basique

Pour créer un ensemble de radio boutons, ajoutez un input avec l'attribut type="radio" et un label correspondant. Réglez l'attribut for de label pour faire correspondre l'id de l'input ainsi ils sont sémantiquement liés.

Groupe vertical

Pour intégrer visuellement de multiples radio boutons dans un ensemble groupé de bouton, le framework va automatiquement supprimer toutes les marges entre les boutons et arrondir seulement les coins supérieurs et inférieurs de l'ensemble s'il y a un attribut data-role="controlgroup" sur le conteneur.

Groupe horizontal

Les radio boutons peuvent également être utilisés pour un ensemble de bouton groupé défini dont uniquement un seul bouton peut être sélectionné à la fois, tels que le gras, l'italique et le soulignement dans les traitements de texte. Pour faire un jeu de bouton horizontale, ajoutez data-type="horizontal" à fieldset.

Taille mini

Pour avoir une version plus compacte, qui est utile dans les barres d'outils et les espaces restreints, ajoutez l'attribut data-mini="true" à l'élément pour créer une version mini.

Position d'icône

Pour changer la position de l'icône du radio qui est par défaut sur ​​la gauche, ajoutez l'attribut data-iconpos="right" au fieldset pour le positionner à droite.

Thème

Pour définir le thème, ajoutez l'attribut data-theme sur chaque case à cocher.
