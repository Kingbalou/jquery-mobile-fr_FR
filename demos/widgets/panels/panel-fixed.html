Positionnement fixe

Il s'agit d'une page typique qui dispose de deux boutons dans la barre d'entête pour ouvrir les panneaux. Le panneau gauche a le mode d'affichage reveal. Le panneau droit s'ouvre avec le mode overlay. Pour les deux panneaux nous définissons data-position-fixed="true". Nous avons également défini une position fixe pour l'entête et le pied de page sur cette page.

Le panneau de gauche contient une longue liste pour démontrer que le framework vérifiera la hauteur du contenu du panneau et "défixera" le panneau de sorte que son contenu puisse défiler. Dans le panneau de droite, il y a un petit formulaire qui montre le positionnement fixe.

Responsive

Pour rendre cela responsive, vous pouvez augmenter la largeur du navigateur. Cela permet à la fois au menu du panneau reveal et à la page d'être utilisés ensemble lorsqu'il y a plus d'espace disponible. Ce comportement est contrôlé par des CSS media queries. Vous pouvez en créer un, sur mesure par un point d'arrêt spécifique ou utiliser le point d'arrêt prédéfini en ajoutant class="ui-responsive-panel" au conteneur de la page. Nous avons ajouté cette classe sur cette page de démonstration.
