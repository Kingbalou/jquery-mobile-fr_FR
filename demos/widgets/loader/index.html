Chargeur API

Une petite superposition du chargeur s'affiche lorsque jQuery Mobile charge dans le contenu via AJAX, ou pour une utilisation dans les notifications personnalisées.

Chargeur standard

La superposition du chargeur peut être uniquement une icône, du texte seul, ou les deux. Ces démonstrations reposent sur un code JavaScript personnalisé, voir la source pour voir comment cela fonctionne.

HTML personnalisé

N'importe quel HTML peut être ajoutée à la superposition du chargeur

Thème

Le nuancier de thème peut être défini pour la superposition du chargeur.
