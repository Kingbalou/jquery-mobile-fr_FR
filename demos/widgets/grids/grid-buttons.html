Boutons dans les grilles

Sur cette page, vous voyez des exemples où vous pouvez utiliser des grilles pour mettre en page les boutons.

Basique

Boutons dans les grilles pour obtenir une petite marge à gauche et à droite.

Icône seule, centrée

Les boutons alignés peuvent être centrés en ajoutant text-align: center; à votre CSS personnalisé.

Responsive

Il n'est pas recommandé d'avoir de nombreux boutons avec du texte sur une ligne sur de petits écrans, parce que le texte pourrait être tronqué. Vous pouvez utiliser les grilles responsive pour empiler les boutons sur de petits écrans. Ici, nous utilisons le point d'arrêt prédéfini du framework en ajoutant la classe ui-responsive au conteneur.
